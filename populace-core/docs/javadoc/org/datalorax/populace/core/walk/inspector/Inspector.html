org.datalorax.populace.core.walk.inspector

Interface Inspector

  • All Known Implementing Classes:
    ArrayInspector, CollectionInspector, FieldInspector, ListInspector, LoggingCollectionInspector, MapValueInspector, SetInspector, TerminalInspector

    public interface Inspector
    Interface for 'pluggable' walking of fields. Inspectors for non-collection types should just implement getFields(java.lang.Class<?>, org.datalorax.populace.core.walk.inspector.Inspectors). Inspectors for collection types e.g. arrays, lists, maps, etc, should provide implementation for getElements(Object, Inspectors). Hybrid inspectors that return both fields and children are supported.
