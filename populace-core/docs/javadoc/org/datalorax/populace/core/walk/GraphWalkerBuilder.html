        public GraphWalkerBuilder withInspectors(Inspectors inspectors)
        Description copied from interface: GraphWalker.Builder
        Install the inspectors that will be used to inspect each object that is encountered when walking the graph. inspectors are used to obtain that set of fields an object has and, for container types, the child objects it contains.

        This call replaces the currently installer set of inspectors. A builder to manipulate the existing Inspectors can be obtained by calling GraphWalker.Builder.inspectorsBuilder()
